# Supported File Types

<p class="lead">Aleph is able to make a wide range of file types searchable. This reference includes a list of file types that Aleph can handle.</p>

<Callout>
  The list of supported file types on this page is not exhaustive.
</Callout>

## Office

| File type | Extensions |
| --- | --- |
| Rich Text Format | `rft` |
| EPUB | `epub` |
| Microsoft Word | `doc`, `docx` |
| OpenOffice/LibreOffice Writer | `odt` |
| Apple Pages | `pages` |
| Microsoft Powerpoint | `ppt`, `pptx` |
| OpenOffice/LibreOffice Impress | `odp` |
| Apple Keynote | `key` |
| PageMaker | `pm` |
| WordPerfect | `wpd` |
| AbiWord | `abw` |

## Documents

| File type | Extensions |
| --- | --- |
| PDF | `pdf` |
| DjVu | `djvu`, `djv` |
| TIFF | `tiff` |

## Plain

| File type | Extensions |
| --- | --- |
| Plaintext | `txt` |
| Markdown | `md` |
| reStructuredText | `rst` |
| HTML | `html`, `htm`, `xhtml` |
| XML | `xml` |
| JSON | `json` |

## Email

| File type | Extensions |
| --- | --- |

| Mbox | `mbox` |

| Outlook Message | `msg` |
| iCaledar | `ics`, `icalendar`, `ical`, `ifb` |
| vCard | `vcf`, `vcard` |
